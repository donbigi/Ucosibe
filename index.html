Case Study Project - Boat Sales

#Data #Visualization #Python #Tableau

1. Background Info

I assumed I work as a data analyst for a yacht and boat sales website. The marketing team is preparing a weekly newsletter for boat owners. The newsletter is designed to help sellers to get more views of their boat, as well as stay on top of market trends.

They want me to look at the recent data and help them learn more about the characteristics of the most viewed boat listings in the last 7 days - is it the most expensive boats that get the most views? Are there common features among the most viewed boats?

Data used for the project was provided by Datacamp download here

The dataset columns details

 

2. Data Validation

Before proceeding, here are the libraries I used in this project.
import numpy as np
import pandas as pd
from IPython.core.display import HTML

It is helpful to comment on everything to avoid getting lost in your code when you get back to viewing the code a month, a year later. Especially useful if you have a terrible memory like me.

The ‘Price’ column contains both the currency and the numeric price. I split the column into two for easy calculations, separating the currency and price.

Splitting column
boat_data = pd.read_csv('boat_data.csv')
boat_data[['Currency', 's_Price']] = boat_data['Price'].str.split(' ', expand=True)

Splitting the price column enabled me to find all distinct currency.

Checking for distinct values
boat_data.Currency.drop_duplicates()

Output shows there are four distinct currencies namely Swiss Franc (CHF), Euro (EUR), Danish Krone (DKK) and British Pound (Â£).

Since there is more than one currency, I converted the currencies to USD to have uniform value for the boat price.

The date of purchase was not listed, so exchange rate used for the conversion was gotten from google currency converter and is valid for January 5, 2022. CHF rate = 1.09, EUR rate = 1.13, DKK rate = 0.15, Pound rate = 1.36.

converting the price to dollars
boat_data['Price_USD'] = np.where(boat_data['Currency'] == 'CHF', boat_data['s_Price'] * 1.09, np.where( boat_data['Currency'] == 'EUR', boat_data['s_Price'] * 1.13, np.where(boat_data['Currency'] == 'DKK', boat_data['s_Price'] * 0.15, boat_data['s_Price'] * 1.36)))

The size of the boats was not provided, but the “length” and “width” columns were provided. To find the size of the boats, I used the Area formula which is length x width.

Creating a column called Area to store the size of the boat
boat_data['Area'] = boat_data['Length'] * boat_data['Width']

Exporting boat_data as CSV
boat_data.to_csv('boat_data_v2.csv', index=False)
